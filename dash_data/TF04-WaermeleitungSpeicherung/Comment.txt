# Anmerkungen zum Testfall

## Probleme bei der Modellierung/Durchführung

In diesem Testfall wird das Durchschwingen von dynamischen Außenlufttemperaturänderungen untersucht.
Damit spielt die thermische Speicherung in den Bauteilen eine erheblich größere Rolle und die Abbildung
von thermischen Speicherkapazitäten wird näher untersucht.
